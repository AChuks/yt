--- title: "Yt::Annotation" h2: "Annotations" ---

Yt::Annotation represents an annotation of a YouTube video.
Initialize Yt::Annotation objects looping through the .annotations collection of each Yt::Video:

{% highlight ruby %} video = Yt::Video.new id: 'BPNYv0vd78A' video.annotations # => # video.annotations.count # => 1 annotation = video.annotations.first # => # {% endhighlight %}

Authentication

All methods of Yt::Annotation do not require any form of authentication.
