--- title: "Comments" h2: "Comments" ---

Yt::CommentThread represents a YouTube comment thread.

Yt::Comment represents a top level comment of a thread or a reply to a comment.

{% highlight ruby %} comment_thread = Yt::CommentThread.new id: 'z13fsrejbwf3dziea22pd5trmyqbjteoa04' comment = Yt::Comment.new id: 'z13fsrejbwf3dziea22pd5trmyqbjteoa04' {% endhighlight %}

Video/Channel Comments

To retrieve all comment threads of a video or channel, you can use associations.

{% highlight ruby %} video = Yt::Video.new(id: 'WW4Bg43afvk') video.comment_threads # => channel = Yt::Channel.new(id: 'UCKG1g_by7n00g3rdjWVFOIQ') channel.comment_threads # => {% endhighlight %}
